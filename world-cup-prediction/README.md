# World Cup Prediction — 五维量化预测框架

基于多维公开数据的足球世界杯冠军及四强深度预测分析技能。专为 Claude Code 设计，通过五个独立维度交叉验证，输出量化概率分布。

## 快速开始
在 Claude Code 对话中直接调用：
/world-cup-prediction
或使用自然语言触发：

> "帮我用五维框架预测2026世界杯冠军和四强"
> "分析一下这届世界杯哪支球队最有可能夺冠"

## 五维分析模型

```
┌─────────────────────────────────────────────────────┐
│                  五维加权预测模型                      │
├──────────┬────────┬─────────────────────────────────┤
│ 维度      │  权重   │ 核心问题                        │
├──────────┼────────┼─────────────────────────────────┤
│ D1 实力   │  30%   │ 真实实力 vs FIFA 排名偏离度？     │
│ D2 数据   │  25%   │ 大数据回测下攻防效率如何？         │
│ D3 结构   │  20%   │ 阵容健康度：年龄/位置/深度？       │
│ D4 历史   │  15%   │ 杯赛基因与心理克制关系？           │
│ D5 环境   │  10%   │ 2026美加墨特殊因素影响？          │
└──────────┴────────┴─────────────────────────────────┘
```

### D1 — 球队实力评分 (Team Strength Rating)

不看 FIFA 排名，用三个子指标动态修正：

- **阵容身价** — Transfermarkt 全队总身价
- **核心球员表现** — 五大联赛 WhoScored/SofaScore 赛季评分
- **战术体系成熟度** — 主教练执教年限 + 近两届大赛战绩

输出：伪强队预警 + 黑马发现

### D2 — 近5万场国际比赛数据模型 (Historical Match Data)

模拟大数据回测，分析过去 3-4 年国际 A 级赛：

- **进攻效率** — xG/90 + 射门转化率
- **防守稳固度** — xGA/90 + 被射门质量
- **逆风球翻盘率** — 先失球后的胜率（心理韧性量化）
- **同级对手胜率** — 对阵 FIFA 前 20 球队的胜率

### D3 — 全队身价结构 (Squad Value Structure)

阵容"健康度"体检：

- **年龄金字塔** — 老(≥30):中(25-29):青(≤24) 黄金比例 20:50:30
- **球星依赖度** — TOP1 球员身价/全队身价，>25% 红色预警
- **板凳深度系数** — 替补 11 人身价/首发 11 人身价，>0.4 为优秀
- **伤病风险** — 近 2 赛季出勤率 <60% 的核心球员标记

### D4 — 世界杯历史交锋 & H2H (World Cup History)

杯赛基因解码：

- **淘汰赛韧性** — 加时赛/点球大战胜率
- **洲际克制关系** — 如巴西对欧洲球队淘汰赛胜率
- **杯赛基因指数** — 淘汰赛胜率×0.4 + 点球胜率×0.2 + 正赛超预选幅度×0.2 + 历史冠军归一化×0.2
- **4×4 H2H 矩阵** — 四强候选相互对战历史

### D5 — 环境适应性与体能负担 (Environment & Physical Load)

2026 美加墨专属分析：

| 因素 | 影响 |
|------|------|
| 跨时区飞行 | 美国跨 4 个时区，考验板凳深度 |
| 墨西哥城高原 (2240m) | 球速加快，高原经验球队受益 |
| 夏季高温高湿 | 美国南部湿度 >70%，身体流占优 |
| 48 队扩军 | 冠军打 8 场 (比往届 +1)，赛程 39 天 |
| 混合草皮 | 部分 NFL 球场改建，MLS 经验者适应更快 |

## 输出格式

每次分析输出包含：

1. **五维加权总分表** — 至少 8 支候选球队，5 个维度分全部显式列出
2. **冠军预测** — 1 支 + 3 条核心理由 + 最大风险点
3. **四强预测** — 4 支球队按夺冠概率排序
4. **黑马预警** — 2-3 支非传统热门 + 晋级路径分析
5. **风险矩阵** — 核心伤病/更衣室/裁判尺度等风险量化

## 设计原则

- **强制约束** — 五维框架不可替换、不可合并、不可跳过
- **数据驱动** — 每个维度必须有量化指标，拒绝纯主观判断
- **独立评分** — 五个维度先独立打分（1-100），再加权合成
- **概率输出** — 加权总分通过 Softmax 换算夺冠概率
- **可证伪** — 所有预测可事后回溯验证
