# World Cup Prediction — 五维量化预测框架

## Overview

对世界杯冠军及四强球队进行深度预测的五维加权分析框架。核心理念：**调取多维公开数据 → 详细测算 → 概率分布**。不依赖单一排名或主观判断，而是通过五个独立维度交叉验证，输出量化概率。

## When to Use

- 用户要求预测世界杯冠军、四强、黑马球队
- 用户要求分析某届世界杯（尤其是2026美加墨）的球队实力
- 用户要求对比多支国家队的大赛竞争力
- 用户询问"哪支球队最有可能夺冠"

## Core Framework: 五维加权模型

**此框架为强制约束，非建议性参考。**

执行分析时，**必须严格按以下五个维度逐一展开**，每个维度给出独立评分（1-100），最后加权合成。**禁止自创其他维度或跳过任何维度。**

**违反框架字面规则就是违反框架精神。** 禁止以"我有更好的分析框架"、"合并维度更高效"、"用户需求不同"等理由偏离五维结构。

### 防偏离锁定

| 禁止行为 | 正确行为 |
|----------|----------|
| 自创7维度/3维度等替代框架 | 严格使用此5维度，可在维度内增加子指标 |
| 声称"综合来看"跳过维度评分 | 每个维度必须先独立评分，再加权合成 |
| 以"数据不足"跳过某维度 | 数据不足时用合理估算 + 标注不确定性 |
| 将维度四(H2H)合并到维度一(实力) | H2H涉及心理/历史因素，必须独立分析 |
| 将维度五(环境)当作注脚一句话带过 | 2026年的地理特殊性必须专项分析 |

### 权重分配

| 维度 | 权重 | 核心问题 |
|------|------|----------|
| 1. 球队实力评分 | **30%** | 真实实力 vs FIFA排名的偏离度？ |
| 2. 近5万场数据模型 | **25%** | 大数据回测下，攻防效率如何？ |
| 3. 全队身价结构 | **20%** | 阵容健康度：年龄/位置/深度？ |
| 4. 世界杯历史交锋 | **15%** | 杯赛基因与心理克制关系？ |
| 5. 环境适应性与体能 | **10%** | 2026美加墨特殊因素影响？ |

## 维度一：球队实力评分 (Team Strength Rating) — 权重30%

**核心指令：** 不要只看FIFA排名。必须综合以下三个子指标给出动态修正分。

### 数据源与子指标

| 子指标 | 数据来源 | 评分方法 |
|--------|----------|----------|
| 阵容身价 | Transfermarkt 全队总身价（欧元） | 48队中排名，归一化到1-100 |
| 核心球员表现 | 五大联赛赛季评分（WhoScored/SofaScore） | 取前5名核心球员平均评分 |
| 战术体系成熟度 | 主教练执教年限、大赛战绩、战术风格稳定性 | 定性评估：执教时长+近2届大赛成绩 |

### 输出要求

- 列出每支候选球队的三个子指标数值
- 给出实力修正分（1-100），并说明相比于FIFA排名的±偏差及原因
- **重点标注：** 谁是被高估的"伪强队"？谁是被低估的黑马？给出判断依据

### 伪强队/黑马判断标准

```
伪强队信号（任一即标注）：
- FIFA排名前15但身价排名低10位以上
- 核心球员＞30岁且无合格替代者
- 近2届大赛均未进八强

黑马信号（任一即标注）：
- FIFA排名15-40但身价排名高10位以上
- 五大联赛核心球员占比＞60%
- 世预赛碾压性表现（净胜球＞+20）
```

## 维度二：近5万场国际比赛数据模型 (Historical Match Data Model) — 权重25%

**核心指令：** 模拟大数据回测。分析各队过去3-4年（涵盖上一届世界杯周期至今）的国际A级赛表现。

### 关键指标

| 指标 | 数据来源 | 分析要点 |
|------|----------|----------|
| 进攻效率 (xG/90) | FBref / Opta / StatsBomb | 场均预期进球，射门转化率 |
| 防守稳固度 (xGA/90) | 同上 | 场均预期失球，被射门质量 |
| 逆风球翻盘率 | 国际A级赛先失球后的胜率 | 心理韧性的量化指标 |
| 同级对手胜率 | 对阵FIFA前20球队的胜率 | 强强对话能力 |

### 输出要求

- 用表格列出各候选球队的四项关键指标
- 标注数据周期（如：2022年12月 - 2026年6月）
- 计算综合效率分：`(xG/90优势 + xGA/90优势 + 翻盘率×2 + 同级胜率×2) / 6` 归一化到1-100
- **重点关注：** 哪些球队的数据存在"水分"（虐菜数据拉高平均值）？

## 维度三：全队身价结构 (Squad Value Structure) — 权重20%

**核心指令：** 分析阵容的"健康度"。不仅看总身价，更要看年龄结构和位置均衡性。

### 分析维度

| 维度 | 评估方法 | 健康阈值 |
|------|----------|----------|
| 年龄结构 | 老(≥30岁):中(25-29岁):青(≤24岁) 身价占比 | 黄金比例 20:50:30 |
| 位置均衡性 | GK/DEF/MID/FWD 四线身价分布 | 无极端偏斜（单线＞50%） |
| 板凳深度 | 替补11人身价总和 / 首发11人身价总和 | 深度系数＞0.4为优秀 |
| 球星依赖度 | TOP1球员身价 / 全队身价 | ＜15%为健康，＞25%为高风险 |

### 输出要求

- 列出每支候选球队的年龄金字塔（三档占比）
- 计算球星依赖度指数，**红色高亮**依赖度＞25%的球队
- 计算板凳深度系数
- **风险预警：** 标记"玻璃人"核心球员（近2赛季出勤率＜60%）及其替代方案评分

## 维度四：世界杯历史交锋 & H2H (World Cup History & Head-to-Head) — 权重15%

**核心指令：** 调取世界杯专属基因。有些球队是"杯赛之王"，有些则是"预选赛龙虫"。

### 分析维度

| 维度 | 数据来源 | 分析要点 |
|------|----------|----------|
| 世界杯历史成绩 | FIFA官方 / RSSSF | 参赛次数、最佳成绩、近5届趋势 |
| 淘汰赛韧性 | 加时赛/点球大战胜率 | 德国、阿根廷的点球传统优势 |
| 洲际克制关系 | 历史H2H统计 | 如：巴西对欧洲球队淘汰赛胜率 |
| 杯赛vs预选赛偏差 | 近3届预选赛成绩 vs 正赛成绩 | 正赛优于预选赛 = 杯赛型球队 |

### 关键H2H矩阵（必须构建）

对于预测的四强候选球队，构建一个4×4的相互对战历史矩阵：

```
         vs球队A   vs球队B   vs球队C   vs球队D
球队A      -        W-D-L     W-D-L     W-D-L
球队B    W-D-L       -        W-D-L     W-D-L
...
```

### 输出要求

- 计算"杯赛基因指数"：`(淘汰赛胜率×0.4 + 点球胜率×0.2 + 正赛超预选幅度×0.2 + 历史冠军数归一化×0.2)`
- 识别潜在的"天敌"克制链（如：A克B、B克C、C克A）

## 维度五：环境适应性与体能负担 (Environment & Physical Load) — 权重10%

**核心指令：** 针对2026美加墨世界杯的特殊性进行专项分析。

### 2026年特殊因素

| 因素 | 影响机制 | 受益方 | 受损方 |
|------|----------|--------|--------|
| 跨时区飞行 | 美国跨4个时区，加拿大跨1.5个 | 板凳深、年轻阵容 | 老将多、轮换少的球队 |
| 墨西哥城高原(2240m) | 空气密度低，球速快，需提前适应 | 南美高原经验球队（玻利维亚/厄瓜多尔/哥伦比亚） | 海平面国家技术流球队 |
| 夏季高温高湿 | 美国南部+东海岸6-7月，湿度＞70% | 身体流/非洲球队 | 北欧/西欧技术流 |
| 48队扩军赛制 | 冠军需打8场（比往届多1场），赛程39天 | 阵容深度＞25人的球队 | 依赖核心首发的球队 |
| 人造草皮/混合草皮 | 部分美式足球场改建 | 有MLS经验球员 | 传统天然草皮联赛球员 |

### 输出要求

- 为每支候选球队计算"环境适应分"：考虑其球员的联赛分布（是否有MLS球员？是否有高原经验者？）
- 根据小组赛赛区分配，评估其旅行负担（飞行距离/跨时区次数）
- 根据球队风格（控球率、跑动距离、身体对抗频率），评估高温高湿影响系数
- **实时更新：** 基于当前小组赛实时积分和伤病情况进行动态调整

## 综合输出格式

完成五维分析后，必须按以下格式输出最终预测。**每个维度的独立评分必须显式展示，不可仅给出加权总分。**

### 1. 五维加权总分表

**至少覆盖8支候选球队**，每支球队的5个维度分必须全部列出：

```
球队      D1(30%)  D2(25%)  D3(20%)  D4(15%)  D5(10%)  加权总分  夺冠概率
法国       92      88       95       85       78       89.1      22%
西班牙     90      91       82       75       72       84.3      18%
英格兰     85      86       90       72       80       83.8      15%
阿根廷     82      84       75       92       85       82.6      13%
德国       80      82       83       88       75       81.4      11%
葡萄牙     83      79       85       70       73       79.5       8%
巴西       78      76       78       90       82       79.0       7%
荷兰       75      80       72       73       77       75.4       4%
...（至少8支）
```

夺冠概率由加权总分通过 Softmax 函数换算（temperature=10）。

**评分一致性检验：** 各维度分数应相互独立——如果某队D1(实力)和D3(结构)同分，需说明是巧合还是有未考虑的关联因素。

### 2. 冠军预测

- 预测冠军（1支）
- 3条核心理由（每条不超过40字）
- 最大风险点

### 3. 四强预测

- 四强名单（4支球队，按夺冠概率排序）
- 每支球队一句话定位

### 4. 黑马预警

- 2-3支可能进入四强的非传统热门球队
- 每支的晋级路径分析

### 5. 风险矩阵

| 风险类型 | 受影响球队 | 影响程度 | 触发条件 |
|----------|-----------|----------|----------|
| 核心伤病 | XXX | 高/中/低 | XXX缺阵 |
| 更衣室矛盾 | XXX | 高/中/低 | 开局不利 |
| 裁判尺度 | XXX | 高/中/低 | 严吹身体对抗 |

## Common Mistakes

| 错误 | 正确做法 |
|------|----------|
| 只看FIFA排名下结论 | 必须用Transfermarkt身价和五大联赛评分做修正 |
| 忽略赛制变化（48队） | 48队扩军后冠军多打1场，阵容深度权重必须上调 |
| 不分析球星依赖度 | 必须计算TOP1球员身价占比，标注＞25%的高风险球队 |
| 忽略2026地理特殊性 | 必须分析墨西哥城高原、美国跨时区、夏季高温的具体影响 |
| 仅用欧洲杯/美洲杯数据替代 | 必须区分"洲际杯表现"与"世界杯表现"，两者相关性约0.6-0.7 |
| 忽略实时数据 | 必须基于当前小组赛实时积分和伤病情况动态修正 |
| 自创维度替代五维框架 | 严禁替换或跳过任何维度；可在维度内增加子指标但不可删除 |

## 自检红旗 — 出现以下任一信号立即回到框架

- "我觉得用自己的维度更合理" → 不可，必须用此框架
- "这个维度没数据，跳过吧" → 不可，估算+标注不确定性
- "H2H和环境放一起说就行了" → 不可，5个维度必须独立展开
- "我合并一下让报告更简洁" → 不可，禁止合并维度
- "用户没要求这么详细" → 不可，框架是最低标准
- "我先给结论再补维度分析" → 不可，维度分析在先，结论在后

**以上任一信号出现 = 正在偏离框架。立即回到五维结构重新输出。**

## 执行顺序约束

分析流程**不可重排**：

```
1. 数据采集（WebSearch/WebFetch 获取各维度最新数据）
      ↓
2. 维度一独立评分（D1: 1-100分 + 伪强队/黑马标注）
      ↓
3. 维度二独立评分（D2: 1-100分 + 数据水分标注）
      ↓
4. 维度三独立评分（D3: 1-100分 + 球星依赖度红色高亮）
      ↓
5. 维度四独立评分（D4: 1-100分 + H2H矩阵 + 杯赛基因指数）
      ↓
6. 维度五独立评分（D5: 1-100分 + 环境适应分）
      ↓
7. 加权合成（五维加权总分表，至少8支球队）
      ↓
8. 最终输出（冠军、四强、黑马、风险矩阵）
```

**不可**先出结论再补维度分析。维度评分必须在综合结论之前独立完成。
